# SmartArt Template Catalog

Complete reference of all 150 unique SmartArt diagram templates available for injection.

**Source:** `templates/smartart/` — 125 variant folders covering 150 unique layout types, each folder containing data.xml, layout.xml, drawing.xml, colors.xml, quickStyle.xml.

## Summary

| Category | Count | Common Use |
|----------|-------|------------|
| cycle | 9 | Circular processes, repeating workflows, continuous cycles |
| hierarchy | 15 | Org charts, decision trees, layered structures |
| list | 28 | Bullet lists, grouped items, tabular displays |
| matrix | 3 | 2×2 grids, quadrant analysis, relationships between axes |
| picture | 25 | Photo grids, image captions, visual showcases |
| process | 37 | Sequential steps, workflows, pipelines, timelines |
| pyramid | 4 | Top-down hierarchy, proportional layers, foundation-based structures |
| relationship | 29 | Venn diagrams, balances, funnels, interconnections |
