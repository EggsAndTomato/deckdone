## Custom Palette Creation Guide

### When to Use a Preset

- Standard presentation types (reports, pitches, reviews)
- Tight timeline — presets are production-ready immediately
- No brand identity constraints
- Audience has no specific visual expectations

### When to Create a Custom Palette

- Brand guidelines mandate specific colors and fonts
- Unique audience or cultural context requiring tailored aesthetics
- Corporate identity compliance is required
- Multiple presentations need a consistent bespoke system

### Steps to Create a Custom Palette

1. **Define 5 core colors** — primary, secondary, accent, background, text. Ensure primary and background have a contrast ratio of at least 4.5:1 (WCAG AA).
2. **Select 2 web-safe fonts** — one for headings (bold personality), one for body (high readability). Stick to Arial, Georgia, Verdana, or Calibri for maximum compatibility.
3. **Test contrast ratios** — verify all text-on-background combinations pass WCAG AA (4.5:1 for body text, 3:1 for large headings).
4. **Document in `style-guide.md`** — record the palette using the same format as the presets above.
